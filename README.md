# AntonMebel - Интернет-магазин мебели

## 📋 Описание проекта

AntonMebel - это веб-сайт интернет-магазина мебели с адаптивным дизайном. Проект представляет собой многостраничный сайт с каталогом товаров, корзиной покупок и информационными страницами.

## 🚀 Функциональность

- **Каталог товаров** с фильтрацией по категориям (стулья, столы, диваны, кресла, шкафы, комоды)
- **Поиск товаров** по названию и описанию
- **Корзина покупок** с возможностью добавления/удаления товаров и изменения количества
- **Детальная информация** о каждом товаре с характеристиками
- **Модальные окна** для просмотра деталей товара и корзины
- **Адаптивный дизайн** для мобильных устройств и планшетов
- **Страницы**: Главная, Каталог, О компании, Оплата

## 🛠️ Технологии

- **HTML5** - структура страниц
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (ES6+)** - интерактивность и логика
- **LocalStorage** - хранение данных корзины

## 💡 Особенности реализации

### Корзина покупок
- Данные хранятся в `localStorage`
- Возможность изменять количество товаров
- Удаление отдельных позиций
- Подсчет общей стоимости
- Модальное окно с анимацией

### Каталог товаров
- Загрузка данных из `products.json`
- Фильтрация по категориям
- Поиск по названию и описанию
- Детальный просмотр товара в модальном окне
- Подсказки с характеристиками при наведении

### Адаптивный дизайн
- Оптимизация для мобильных устройств (до 480px)
- Планшетная версия (до 768px)
- Десктопная версия (более 1024px)

## 🎨 Дизайн

- Цветовая схема: синий (`#0a4788`) и белый
- Шрифты: Tahoma, Verdana
- Анимации: плавные переходы, всплывающие подсказки
